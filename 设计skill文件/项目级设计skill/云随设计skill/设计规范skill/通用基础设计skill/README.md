# quick-use 正式强约束投喂包

本目录是正式项目推荐入口。它的约束力强于 `../daily-use/`，因为生成/审核流程文件会和完整 `04` 设计规范约束基座一起投喂。

## 推荐组合

| 场景 | 投喂方式 |
| --- | --- |
| PC Demo 生成前 | 产品文档 + `01A-PC-Demo生成前-文档输入版-发给IDE-Agent.md` + `04-设计规范约束速查-给AI一起看.md` |
| App/H5/小程序 Demo 生成前 | 产品文档 + `02A-App-Demo生成前-文档输入版-发给IDE-Agent.md` + `04-设计规范约束速查-给AI一起看.md` |
| Demo 生成后审核 | Demo 材料 + 产品文档 + `03A-Demo生成后-AI审核-文档输入版-发给IDE-Agent.md` + `04-设计规范约束速查-给AI一起看.md` |

## 文件说明

| 文件 | 作用 |
| --- | --- |
| `00-总介绍-先看这个.md` | 总说明 |
| `01A-PC-Demo生成前-文档输入版-发给IDE-Agent.md` | PC 生成前流程 |
| `02A-App-Demo生成前-文档输入版-发给IDE-Agent.md` | App 生成前流程 |
| `03A-Demo生成后-AI审核-文档输入版-发给IDE-Agent.md` | 审核流程 |
| `04-设计规范约束速查-给AI一起看.md` | 完整设计规范约束基座 |

`daily-use/` 是 Lite 试用视图，适合快速试阅；正式项目默认使用本目录。
