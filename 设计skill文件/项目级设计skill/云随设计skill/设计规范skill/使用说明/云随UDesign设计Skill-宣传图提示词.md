# 云随 UDesign 设计 Skill 宣传图提示词

生成一张“云随 UDesign 项目级设计 Skill”竖版宣传图，面向正在处理云随需求的产品经理和研发，重点表达：只需把云随 Skill 与需求文档、业务诉求或测试数据一起交给 AI，AI 即可复用设计师已沉淀的云随业务规则、正式 Figma 样板、成熟页面模式和 UDesign 组件规范，自动补齐关键状态并生成可直接打开的单 HTML 页面 Demo；产品或研发验证后可直接反馈问题，若需求涉及 Skill 尚未覆盖的复杂核心页面，AI 应输出设计缺口，由设计师补充 Figma 样板并回灌 Skill 后再生成。画面采用简洁专业的蓝白科技风，突出“需求 + Skill → AI 生成 Demo → 产品/研发验证 → 问题与设计缺口回流”的闭环，减少装饰和小字，确保信息一眼可读，不出现“产品/研发先判断 L0～L3”“自行寻找 Figma 地址”或“AI 替代设计师”等表述。
