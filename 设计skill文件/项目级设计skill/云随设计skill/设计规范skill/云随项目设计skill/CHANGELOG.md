# Changelog

## 0.1.2

- 内置 UDesign Empty 官方通用状态 SVG。
- 新增项目插图与空状态兜底规则。
- 明确普通空状态缺少 Figma 插图时不阻断 Demo 生成，可使用 UDesign Empty 或云随中性占位。
- 明确只有插图承载核心业务含义、风险解释或强品牌主视觉时，才输出设计缺口。
- 更新生成、评审、交付和使用说明中的插图检查项。

## 0.1.1

- 内置 UDesign / TDesign Icons 官方 SVG 图标库快照。
- 新增图标来源、选择、单 HTML 内联和评审规则。
- 明确禁止使用 emoji、第三方图标库或 AI 自行手写 SVG 作为云随正式图标。
- 更新生成与评审提示词，要求图标来自 Skill 内置官方资源。

## 0.1.0

- 完成云随项目级设计 Skill 第一版。
- 建立设计师建设与产品/研发日常使用两个阶段。
- 建立正式 Figma、UDesign、云随成熟模式与云随内部补全策略的使用边界。
- 建立成熟页面生成、多状态组合和设计缺口回流机制。
- 完成公司 CC + GLM-5.2 环境下六类场景的第一轮可行性验证。
- 当前版本进入业务试用准备阶段。
