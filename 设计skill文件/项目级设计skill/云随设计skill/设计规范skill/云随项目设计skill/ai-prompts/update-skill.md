# 更新 Skill 提示词模板

适用对象：设计师维护云随 Skill 时使用。

```text
请基于以下试用反馈评估是否需要更新 $yunsui-design。

## 试用反馈
[粘贴反馈记录、问题描述、Demo 文件、截图或评审结论]

请先判断问题属于哪一类：

- 需求输入不足。
- Skill 知识缺失。
- Figma 样板缺失。
- UDesign 映射缺失。
- AI 执行错误。
- 单次特殊需求。
- 可沉淀的重复稳定模式。

请给出：

1. 是否需要更新 Skill。
2. 若不更新，应该如何修改本次 Demo 或补充需求。
3. 若需要更新，应修改哪些文件。
4. 是否需要设计师先补充正式 Figma 样板。
5. 是否需要补充 UDesign 映射或云随内部补全策略边界。
6. 更新后建议使用哪些真实需求回归测试。

只有重复出现、形成稳定模式、正式业务规则变化或新增正式 Figma 样板时，才更新 Skill。不要把单次特殊需求写入通用规则。
```
