# UDesign Empty 状态插图来源记录

本目录保存 UDesign / TDesign React `Empty` 组件的官方默认状态 SVG，用作云随 Demo 在缺少正式 Figma 插图时的通用兜底。

## 来源

- 官方仓库：https://github.com/Tencent/tdesign-react
- 组件目录：`packages/components/empty`
- 下载日期：2026-07-15
- 快照 commit：`9fae989e9bdf13234ec0958f786d62aba3af5f2a`
- 授权协议：MIT，见 [LICENSE](LICENSE)

## 本地内容

- `empty.svg`：无数据 / 暂无结果。
- `success.svg`：成功 / 已完成。
- `fail.svg`：失败 / 执行失败。
- `network-error.svg`：网络或系统连接异常。
- `maintenance.svg`：维护中 / 暂不可用。
- `illustration-index.json`：状态插图索引。

## 使用边界

这些 SVG 是 UDesign 通用状态图，不是云随项目级品牌插图。它们只用于没有正式 Figma 或项目插图资产时，保持 Demo 完整性和基础一致性。

禁止将这些通用状态图包装成云随正式主视觉；若页面需要强品牌表达，应在交付说明中标记“项目插图待补充”。
